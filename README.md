#sqzr-Blog

## 简介
基于struts2 + mybatis + spring 开发的采用Markdown语法发布文章的博客程序,后台UI采用ace Bootstrap模板,目前只是简单实现了基本的文章分类的增删改以及博客基础信息的修改

------

## 开发目的
一直没有项目经验,[自己博客](http://sqzr.cc) 一直采用octpress静态生成,想添加一点功能又不懂ruby.自学完java基础后,就开始动手构建一个这样的博客程序

------
